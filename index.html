Beautiful Design: Inside and Out

This website was built using the latest in HTML5 technologies and kept lightweight without javascript. If you would like to find out more about this site you can visit the GitHub repository where it is stored or contact me.

Semantic Markup

The point of semantic markup is to make the interals of a site less unsightly. Without looking at the rendered page, sematics make it easy to figure out what the HTML is doing. These semantic markups pair well with the existing <h1> <h2>... tags to provide and outline of the page along with level of importance. For example, you are in an <article> right now!

CSS3

This site was made possible with new CSS3 specifications. Here a couple just to name a few:

  • :target selectors were used to move from one piece of content to then next without leaving the page.
  • Transitions provided the user with an overall smoother experience on the site.
  • background-size ensured that the images fit appropriately no matter the screen size or orientation.

Sass

Sass isn't part of HTML5 but it should be. It is a css extension that provides developers and designers alike the power of reusable code inside to generate normal css stylesheets.
